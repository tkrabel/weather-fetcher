# Metadaten – wetterdaten.csv

Wetterstation: **IGREIF68** (Weather Underground Personal Weather Station)
API: `api.weather.com/v2/pws/history/daily` · Einheiten: metrisch

---

| Spalte | Beschreibung | Einheit |
|---|---|---|
| `Datum` | Datum der Messung (Ortszeit) | YYYY-MM-DD |
| `Temp_Hoch_°C` | Höchsttemperatur | °C |
| `Temp_Avg_°C` | Durchschnittstemperatur | °C |
| `Temp_Tief_°C` | Tiefsttemperatur | °C |
| `Taupunkt_Hoch_°C` | Höchster Taupunkt | °C |
| `Taupunkt_Avg_°C` | Durchschnittlicher Taupunkt | °C |
| `Taupunkt_Tief_°C` | Niedrigster Taupunkt | °C |
| `Windchill_Hoch_°C` | Höchste gefühlte Temperatur (Windchill) | °C |
| `Windchill_Avg_°C` | Durchschnittliche gefühlte Temperatur (Windchill) | °C |
| `Windchill_Tief_°C` | Niedrigste gefühlte Temperatur (Windchill) | °C |
| `Hitzeindex_Hoch_°C` | Höchster Hitzeindex (gefühlte Temperatur bei Wärme und Feuchtigkeit) | °C |
| `Hitzeindex_Avg_°C` | Durchschnittlicher Hitzeindex | °C |
| `Hitzeindex_Tief_°C` | Niedrigster Hitzeindex | °C |
| `Luftfeuchtigkeit_Hoch_%` | Höchste relative Luftfeuchtigkeit | % |
| `Luftfeuchtigkeit_Avg_%` | Durchschnittliche relative Luftfeuchtigkeit | % |
| `Luftfeuchtigkeit_Tief_%` | Niedrigste relative Luftfeuchtigkeit | % |
| `Windgeschwindigkeit_Hoch_km/h` | Höchste Windgeschwindigkeit | km/h |
| `Windgeschwindigkeit_Avg_km/h` | Durchschnittliche Windgeschwindigkeit | km/h |
| `Windgeschwindigkeit_Tief_km/h` | Niedrigste Windgeschwindigkeit | km/h |
| `Windböe_Hoch_km/h` | Höchste Windböe | km/h |
| `Windböe_Avg_km/h` | Durchschnittliche Windböe | km/h |
| `Windböe_Tief_km/h` | Niedrigste Windböe | km/h |
| `Windrichtung_Avg_°` | Durchschnittliche Windrichtung (0° = Nord, 90° = Ost, 180° = Süd, 270° = West) | ° |
| `Luftdruck_Max_hPa` | Maximaler Luftdruck | hPa |
| `Luftdruck_Min_hPa` | Minimaler Luftdruck | hPa |
| `Luftdruck_Trend_hPa` | Luftdruckveränderung im Tagesverlauf | hPa |
| `Sonnenstrahlung_Hoch_W/m2` | Höchste Sonneneinstrahlung | W/m² |
| `UV_Index_Hoch` | Höchster UV-Index (Skala 0–11+) | Index |
| `Niederschlag_Total_mm` | Gesamtniederschlag des Tages | mm |
| `Niederschlagsrate_Max_mm/h` | Maximale Niederschlagsintensität | mm/h |
| `QC_Status` | Qualitätskontrollstatus der Messung (0 = nicht geprüft, 1 = bestanden) | — |
